# 学習サポートアプリ
## DB情報
### Table: info_account
#### Columns
- user_name(varchar): ユーザー名
- login_id(varchar): ログインID
- user_pass(varchar): パスワード
- table_id(int): ユーザー識別番号
- memo(varchar): メモ帳内容
- countdown_title(varchar): カウントダウン(タイトル)
- countdown_date(date): カウントダウン(yyyy-MM-dd)
